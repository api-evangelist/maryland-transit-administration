# Maryland Transit Administration (maryland-transit-administration)
The Maryland Transit Administration (MTA) supports open transit data initiatives and makes resources available to developers and applications. It primarily uses the General Transit Feed Specification (GTFS) to convey schedule, geographic, fare, and other related datasets in a standardized format.
